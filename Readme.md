# xLib - C++ cross-platform library

Authors: skynowa

E-mail:  skynowa@gmail.com

Skype:   skynowa777

## Features

- platforms:

    - Windows: (XP >)
    - Unix: Linux, Android, BSD, Apple (OSX)

- architecture: 32-bit (x86) and 64-bit (x86-64, IA64, and AMD64)
- compilers: MinGW, MSVC, GCC, Clang
- unicode support
