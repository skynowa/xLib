## Features

- Language: C++ 20
- Platforms:

  - Windows (XP >)
  - Unix:
      - Ubuntu 20.04
      - BSD
      - Apple (OSX)

- Compilers:
    - GCC
    - Clang
    - MSVC
    - MinGW

## Pre-requisites

- bash-compatible Unix command shell (for Unix)
- Doxygen 1.8.2: `sudo apt-get install doxygen`
- `sudo apt-get install cmake pkg-config`
- `~/Projects/CMakeLib/install.sh`
